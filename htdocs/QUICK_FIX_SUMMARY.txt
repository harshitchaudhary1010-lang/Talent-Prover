TROUBLESHOOTING:
================================================================================

Problem: Apache won't start
Solution: Port 80 is in use
   - Change Apache to port 8080 in XAMPP config
   - Then use: http://localhost:8080/
   - OR stop Skype/IIS/other services using port 80

Problem: MySQL won't start
Solution: Port 3306 is in use
   - Change MySQL port in XAMPP config
   - Update config/db.php with new port

Problem: Still getting 404 errors
Solution: Check these in order:
   1. Apache is running (green in XAMPP)
   2. .htaccess file exists in c:\xampp2.0\htdocs\
   3. mod_rewrite is enabled in Apache config
   4. Try with .php extension: http://localhost/index.php

Problem: Database connection failed
Solution: 
   1. MySQL is running (green in XAMPP)
   2. Check config/db.php settings:
      - Host: localhost
      - User: root
      - Password: (empty)
      - Database: talentprover

Problem: Blank white page
Solution: You're opening file directly
   - Use http://localhost/ not file:///
   - Make sure Apache is running first
