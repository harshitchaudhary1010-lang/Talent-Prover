🗓️ Test Expired Tasks Feature

Follow these steps to test the new expired task blocking feature

📋 What This Feature Does:

Students can no longer view or submit to tasks that have passed their deadline date. Tasks are automatically filtered based on today's date.

  • Expired tasks are NOT deleted, just hidden from students

  • The system uses server date (today's date) for comparison
  • Date comparison: deadline >= CURDATE()
