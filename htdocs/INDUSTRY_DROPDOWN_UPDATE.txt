HOW IT WORKS:
================================================================================

STEP 1: User opens company registration
   → Go to: http://localhost/auth/register.php?role=company

STEP 2: User fills company details

STEP 3: Industry dropdown shows 4 options:
   ┌─────────────────────────┐
   │ Select industry...      │
   │ Technology             │
   │ Finance                │
   │ Healthcare             │
   │ Other (Specify)        │ ← Click this for custom
   └─────────────────────────┘

STEP 4a: If user selects Technology/Finance/Healthcare
   → Value is saved as-is (already capitalized)

STEP 4b: If user selects "Other (Specify)"
   → New text field appears below
   → User types custom industry (e.g., "education")
   → First letter auto-capitalizes → "Education"
   → Final value: "Education"

TECHNICAL DETAILS:
================================================================================

Files Modified:
   - auth/register.php (HTML structure + JavaScript)

Changes Made:
   1. Replaced text input with <select> dropdown
   2. Added predefined options (capitalized)
   3. Added "Other" option
   4. Added hidden custom text field
   5. Added JavaScript to show/hide custom field
   6. Added JavaScript to capitalize first letter
   7. Added hidden input to store final industry value
   8. Added form validation for industry

HTML Structure:
   <select name="industry_select">
       <option value="Technology">Technology</option>
       <option value="Finance">Finance</option>
       <option value="Healthcare">Healthcare</option>
       <option value="Other">Other (Specify)</option>
   </select>
   
   <input name="industry_custom" /> <!-- Shows when Other selected -->
   <input name="industry" type="hidden" /> <!-- Final value sent to API -->

JavaScript Logic:
   - onChange of dropdown → check if "Other" selected
   - If "Other" → show custom text field
   - On form submit → check which option selected
   - If "Other" → capitalize custom value → save to hidden field
   - If predefined → save dropdown value directly
   - Send hidden field value to API

================================================================================
CAPITALIZATION LOGIC:
================================================================================

Method: JavaScript charAt(0).toUpperCase()

Examples:
   "technology"     → "Technology"
   "finance"        → "Finance"  
   "education"      → "Education"
   "real estate"    → "Real estate"
   "e-commerce"     → "E-commerce"
   "MANUFACTURING"  → "MANUFACTURING" (keeps if all caps)

Only first character is capitalized, rest remains as typed.
